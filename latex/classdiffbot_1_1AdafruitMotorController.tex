Set the speed of the motor p\+Motor\+\_\+. 

Concrete implementation of the set\+Speed interface method to control a single motor connected to the Adafruit\+\_\+\+Motor\+Shield. The input parameter {\ttfamily value} ranges from -\/255 to 255. Inside this method the value is mapped between 0 to 255 and the sign is used to set the direction. A positive {\ttfamily value} results in the motor spinning forward. Negative {\ttfamily value} rotates the motor backward and a zero value stops the motor (releases the current).
