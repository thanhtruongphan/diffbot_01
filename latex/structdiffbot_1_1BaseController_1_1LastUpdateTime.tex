Keeps track of the last update times. 

Inside the \mbox{\hyperlink{main_8cpp}{main.\+cpp}} the members of this \mbox{\hyperlink{classdiffbot_1_1BaseController_a1387b4e876fabe7bee2651f7bde4056e}{last\+\_\+update\+\_\+time\+\_\+}} are updated everytime the corresponding event happens. This is used together with the \mbox{\hyperlink{classdiffbot_1_1BaseController_a0dad88e3a39fd4a5cfa83eef522fa948}{update\+\_\+rate\+\_\+}} to check if enough time elapsed for a new update. For example reading new values from the encoders and writing the received motor commands.
